# Gestion bibliothèque
Ceci est un exemple académique d'un mini projet de gestion de bibliothèque, ce dépot contient 3 projets java, le premier projet (ConceptionDAO) contient le métier du projet et les interfaces de la couche dao et la classe qui permet la connection avec la base de données. le second projet (DAOImplements) contient les classes d'implimentation des interfaces de la couche dao et le troixième projet(Bibliothèque) contient les controlleurs et les views de tout le projet
# JarFiles
Le répertoire (JarFiles) contient les fichiers .JAR de tout le projet.
